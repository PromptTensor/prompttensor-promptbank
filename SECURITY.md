# Security Policy

## Supported Versions
This project is a public dataset repository. Security-related changes will be released as updates to the repository.

## Reporting a Vulnerability
If you discover a security issue (e.g., exposed secrets, malicious files, or unsafe content), please report it privately:

- Email: security@prompttensor.com (or your preferred security inbox)
- Please include: a clear description, steps to reproduce (if applicable), and links/paths to affected files.

We will acknowledge receipt and work on a fix as soon as possible.
